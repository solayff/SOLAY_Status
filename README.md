# SOLAY_Status

Self-hosted Uptime Monitoring Tool  
Powered by [Uptime Kuma](https://github.com/louislam/uptime-kuma) — customized and deployed by @solayff

> Monitor your websites, APIs, and services with style and full control.

## Features
- 🟢 Website monitoring (HTTP, TCP, Ping, etc.)
- 📊 Beautiful dashboard
- 💬 Telegram, Discord, Email notifications
- 🌍 Public Status Page
- 🔐 Self-hosted and secure

## Deployment
Deployed on [Fly.io](https://fly.io) using GitHub + CI/CD pipeline.
